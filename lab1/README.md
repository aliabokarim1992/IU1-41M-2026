# Задание №1

## Цель задания

- Освоить базовые понятия машинного обучения.
- Настроить пайплайн работы с Git.

## Ключевые навыки

- Генерация случайной выборки, борьба с переобучением.

## Задание

1. Загрузите датасет: https://archive.uci.edu/dataset/53/iris
2. Постройте таблицу **pairplot** для визуального анализа данных.
3. Постройте **LogisticRegression** для бинарной классификации элементов датасета.
4. На примере данных *(можно самостоятельно сгенерировать)* покажите переобучение модели линейной регрессии.

---

## Сдача работы

Каждый студент создаёт внутри папки `lab1` папку со своим **именем и фамилией на латинице** в едином формате:

```
lab1/
└── firstname_lastname/   # например: ivan_petrov
    ├── report.pdf        # PDF-отчёт
    └── notebook.ipynb    # Jupyter Notebook с кодом
```

В папку помещаются **два файла**:
- `report.pdf` — письменный отчёт в формате PDF.
- `notebook.ipynb` — Jupyter Notebook с выполненным кодом.
