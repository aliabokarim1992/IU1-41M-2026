# Задание №2

## Цель задания

- Обработка кардиологического датасета для решения задач бинарной классификации.

## Ключевые навыки

- Подготовка данных, применение **t-SNE** и **PCA**.
- Применение **AutoML**-моделей, расчёт точности классификатора.

## Задание

1. Загрузить **5000 строк** из датасета *(см. файлы с названием «модуль 3…»)*:
   https://github.com/AI-is-out-there/data2lab.git

2. Сформировать обучающую выборку из следующих столбцов:
   ```
   ['Count_subj', 'rr_interval', 'p_end', 'qrs_onset', 'qrs_end',
    'p_axis', 'qrs_axis', 't_axis', 'Healthy_Status']
   ```

3. Датасет состоит из числовых параметров ЭКГ и классификационного признака **Healthy_Status**.

4. Исследуй фреймворки **AutoML**, обоснуй выбор лучшего. Используй его для бинарной классификации.

5. Построй матрицу ошибок (**confusion matrix**) и рассчитай **F1-метрику** для оценки обученного классификатора по признаку `Healthy_Status` на основе параметров ЭКГ.

## Сдача работы

Каждый студент создаёт внутри папки `lab2` папку со своим **именем и фамилией на латинице** в едином формате:

```
lab2/
└── firstname_lastname/   # например: ivan_petrov
    ├── report.pdf        # PDF-отчёт
    └── notebook.ipynb    # Jupyter Notebook с кодом
```

В папку помещаются **два файла**:
- `report.pdf` — письменный отчёт в формате PDF.
- `notebook.ipynb` — Jupyter Notebook с выполненным кодом.
