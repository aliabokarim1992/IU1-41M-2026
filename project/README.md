# Проектная сессия — Группа ИУ1И-41М

## Аналитическое исследование медицинских изделий с ИИ

1. Зайди на гугл-файл:
   https://docs.google.com/spreadsheets/d/1-iSg1FfCxplSJHf80IRggm-X9ECG3IH8QVKdLyiGTo4/edit?usp=sharing

2. Найди свою фамилию.

3. Перейди по ссылке продукта **K252054**.

4. Откроется описание продукта — найди раздел **Summary**.

5. Перейди по ссылке Summary (откроется подробное описание).

6. В описании найди номер **Predicate Device** *(например: K243623)*.

7. Вставь найденный номер в гугл-файл в столбец «Predicate Device».

8. Напиши аналитический отчёт по итогам исследования.

## Сдача работы

Каждый студент создаёт внутри папки `project` папку со своим **именем и фамилией на латинице** в едином формате:

```
project/
└── firstname_lastname/   # например: ivan_petrov
    └── report.pdf        # PDF-отчёт
```

В папку помещается **один файл**:
- `report.pdf` — письменный отчёт в формате PDF.
